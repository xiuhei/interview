# React 渲染机制 的核心概念是什么？

## question
请解释 React 渲染机制 的核心概念，并说明它主要解决什么问题。

## reference_answer
React 渲染机制 是 Web 前端 面试里常见的高频知识点，回答时应覆盖定义、机制、适用场景、边界条件和代价。 高分回答要先给结论，再说明它解决的问题、适用边界和真实使用场景。

## key_points
- 先讲清 React 渲染机制 的定义或问题背景
- 再说明机制、限制和真实场景
- 补充项目证据、指标或验证方式
- 明确边界条件与风险点

## common_mistakes
- 只背结论，不解释为什么
- 不说明适用边界和代价
- 无法给出真实项目或验证证据
