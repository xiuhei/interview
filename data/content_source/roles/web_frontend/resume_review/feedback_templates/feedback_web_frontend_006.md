# 反馈模板：结果导向反馈

## template
结果导向反馈：先肯定真实经验，再指出缺口，最后给出一条可执行修改建议。

## when_to_use
适用于简历评分结果需要解释和安抚，同时推动候选人继续优化的场景。

## example
你在 Web 前端 方向上已经有一定项目素材，但当前最影响说服力的是职责边界和结果量化，建议优先补这一层。
