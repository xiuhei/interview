# Web Vitals：场景应用

## question
Web Vitals 在真实项目里适合怎么用？

## answer
Web Vitals 是 Web 前端 面试里常见的高频知识点，回答时应覆盖定义、机制、适用场景、边界条件和代价。 回答时要强调定义、机制、场景和边界，而不是只背结论。

## extended_explanation
围绕 Web Vitals 继续展开时，应补充真实项目中的收益、代价、监控方式和不适用场景。
