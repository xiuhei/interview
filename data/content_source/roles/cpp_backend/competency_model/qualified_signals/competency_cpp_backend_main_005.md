# 常见不达标表现说明

## summary
C++ 后端 在 常见不达标表现说明 场景里，更关注候选人是否能把知识、工程判断、项目证据和结果闭环说清楚。

## signals
- 能围绕 C++ 语言基础 给出机制解释、场景取舍和可验证结果
- 能围绕 操作系统与网络 给出机制解释、场景取舍和可验证结果
- 能围绕 算法与数据结构 给出机制解释、场景取舍和可验证结果
- 能围绕 系统设计 给出机制解释、场景取舍和可验证结果

## interview_focus
- 定义是否准确
- 取舍是否自洽
- 是否有真实项目证据
- 是否能解释风险与边界

## usage
这些文档主要用于面试评分、简历评分和成长建议的解释层。
