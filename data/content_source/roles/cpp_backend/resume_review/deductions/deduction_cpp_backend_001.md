# 常见扣分项：只罗列技术栈，没有职责和结果

## diagnosis
只罗列技术栈，没有职责和结果

## impact
这类问题会降低 C++ 后端 简历的可信度、岗位匹配度或成长潜力判断。

## fix
优先补齐职责边界、技术动作、结果指标和验证方式。

## reviewer_comments
扣分项文档主要用于解释为什么系统给出低分。
