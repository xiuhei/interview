# API 概览

## 认证
- `POST /api/v1/auth/register`
- `POST /api/v1/auth/login`
- `GET /api/v1/auth/me`

## 岗位与知识库
- `GET /api/v1/positions`
- `GET /api/v1/positions/{code}/competencies`
- `GET /api/v1/knowledge-base/status`
- `POST /api/v1/knowledge-base/rebuild`

## 简历
- `POST /api/v1/resumes/upload` `multipart/form-data`
- `POST /api/v1/resumes/{resume_id}/parse`
- `GET /api/v1/resumes/{resume_id}/summary`

## 面试会话
- `POST /api/v1/interviews`
- `GET /api/v1/interviews/{session_id}/first-question`
- `POST /api/v1/interviews/audio/upload`
- `POST /api/v1/interviews/{session_id}/answers`
- `GET /api/v1/interviews/{session_id}/next-question`
- `GET /api/v1/interviews/{session_id}`
- `POST /api/v1/interviews/{session_id}/complete`
- `GET /api/v1/interviews/{session_id}/report`

## 记录与成长
- `GET /api/v1/records/interviews`
- `GET /api/v1/records/reports/{session_id}`
- `GET /api/v1/growth/trends`
- `GET /api/v1/growth/weaknesses`
- `GET /api/v1/growth/plan`

## 调试
- `GET /api/v1/debug/retrievals/{session_id}`
- `GET /api/v1/debug/prompts/{session_id}`
- `GET /api/v1/debug/scoring/{answer_id}`

所有接口统一返回：
```json
{
  "code": 0,
  "message": "ok",
  "data": {},
  "request_id": "..."
}
```
